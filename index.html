从零开始的 Hexo 博客搭建指南

前言

你是否也曾想拥有一个属于自己的博客？无论是记录技术笔记、分享生活感悟，还是打造个人品牌，Hexo 都是一个绝佳的选择。

本文将带你从零开始，一步步搭建并部署一个 Hexo 博客。

什么是 Hexo？

Hexo 是一个快速、简洁且高效的静态博客框架，基于 Node.js 构建。你只需用 Markdown 撰写文章，Hexo 就能在几秒内生成带有精美主题的静态网页[reference:0]。

核心优势：

  • ⚡ 极速生成：数百篇文章秒级渲染
  • 🎨 海量主题：社区拥有超过 400 款主题
  • 📦 一键部署：支持 GitHub Pages、Vercel 等多种平台

环境准备

在开始之前，请确保你的电脑已安装以下工具：

1. 安装 Node.js

访问 Node.js 官网 下载并安装 LTS 版本。安装完成后，打开终端验证：

node -v
npm -v-
